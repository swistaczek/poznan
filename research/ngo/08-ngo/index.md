# ngo / 08 — baza NGO

**TL;DR.** Baza ~15 organizacji z realną sprawczością w obszarach repo `poznan`: watchdog (Watchdog Polska, Panoptykon), pro bono prawne (HFPC), hałas (Instytut Ekologii Akustycznej / ORION — kluczowy dla Dąbrowskiego), smog (PAS), mobilność (Rowerowy Poznań, ECF), urbanistyka (KRM + poznańscy członkowie: Prawo do Miasta, Ulepsz Poznań), partycypacja/granty (Batory/Masz Głos, Stocznia, Fundusz Obywatelski im. Wujców, PISOP, Akademia Dialogu), media lokalne (Tomasz Cylka, Radio Poznań). Model operacyjny: „Hiperkoordynacja Dźwigniowa" — repo nie tworzy nowych ciał eksperckich, tylko przejmuje wiedzę i potencjał procesowy gigantów. Każdy kontakt musi mieć formę ustrukturyzowanego briefu prawnego, inaczej zostanie zakwalifikowany jako „szum informacyjny".

## Szablony pism

Brak gotowych szablonów pism w źródle. Oryginał wskazuje zewnętrzne źródła wzorów:

- **Instytut Ekologii Akustycznej** — zakładka „PISMA" na https://instytut.ngo/ (sprofesjonalizowane wzory dla spraw hałasowych).
- **Watchdog Polska** — portal `porady.siecobywatelska.pl` (indywidualna praca na sprawie, nie gotowe szablony).
- **KRM „Anty-Bezradnik przestrzenny"** — procedury dla MPZP/Studium.

Kiedy powstaną wewnętrzne szablony repo → `/szablony/ngo/`.

## Checklist pierwszego kontaktu (z [01-meta-taktyka](01-meta-taktyka.md))

1. Zwięzłość: stan faktyczny w 4–5 zdaniach, precyzyjne daty, nazwy organów, numery decyzji.
2. Udokumentowana ścieżka (wyczerpane wcześniejsze kroki).
3. Precyzyjne oczekiwanie: porada / wzór skargi do WSA / amicus curiae.
4. RODO — anonimizacja dokumentów.
5. Kanał dedykowany, nie ogólne `biuro@...`.
