# Hałas ul. Dąbrowskiego — retrospektywa 2016-2026

> Źródło: [`../wyniki-halas-dabrowskiego-historia.md`](../wyniki-halas-dabrowskiego-historia.md)

## TL;DR (3 zdania)

ZDM Poznań dysponował od 2016 r. pomiarem LAeqN = 62,2 dB przy Roosevelta 11/Dąbrowskiego (punkt 17, 211 tramwajów/noc, v = 15,3 km/h) — maksimum dla całego badania tramwajowego miasta, przekroczenie normy 56 dB o 6,2 dB. Reakcja infrastrukturalna (ciche torowisko 1,5 km Żeromskiego–Most Teatralny, wymiana rozjazdów Kraszewskiego, FEW 2021-2027 ok. 51 mln zł) nastąpiła dopiero w latach 2024-2026, po dekadzie ekspozycji tysięcy mieszkańców na przekroczenia generujące NHA/NHSD/NIHD. W tym czasie Rozporządzenie MŚ z 1.10.2012 r. (Dz.U. poz. 1109) zliberalizowało limity nocne o ok. 10 dB, a MPZP Jeżyc przerzucił koszt ochrony biernej (stolarka Rw > 40 dB, nawiewniki) na prywatne budżety mieszkańców.

## Oś czasu

| Rok | Zdarzenie |
|---|---|
| 2007 | Rozp. MŚ z 14.06.2007 (Dz.U. 120 poz. 826) — normy referencyjne dla pomiarów 2016: LAeqD = 65 dB, LAeqN = 56 dB dla zabudowy mieszkaniowej |
| 2012 | **Liberalizacja norm** (Rozp. MŚ z 1.10.2012 r., Dz.U. poz. 1109) — +10 dB w nocy dla zabudowy śródmiejskiej (50–55 → 60 dB) |
| 2014 | Tekst jednolity (Dz.U. 2014 poz. 112) |
| 2016 | Pomiary ZDM: 164 punkty (37 tramwajowych), 62,2 dB w pkt 17 — diagnoza publiczna |
| 2017 | Inicjatywa „Stop hałasowi w Poznaniu!" (kamienica Dąbrowskiego 96) |
| 2018 | **Uchwała RM LXIX/1253/VII/2018** — pierwszy POŚpH dla Poznania |
| 2021 | Straż Miejska: 62 kontrole hałasu (wnioski do sądu) + 116 kontroli / 5 mandatów |
| 2022 | Strategiczna Mapa Hałasu (AKUSTIX + Lemitor) — LDWN 73–67 dB, LN 65–60 dB na Dąbrowskiego |
| 2024-07-15 | **Uchwała Sejmiku IV/92/24** — PHNUA wojewódzki (obejmuje Poznań, zastępuje 2018) |
| 2024-07-18 | Utrata mocy uchwały RM 2018 (zmiana POŚ — kompetencja Marszałka) |
| 2025 (marzec) | Oddanie odcinka Dąbrowskiego przy Kochanowskiego (nowe przejścia, nasadzenia) |
| 2025-2026 | „Ciche torowisko" 1,5 km Żeromskiego → Most Teatralny (maty PU, TAT, wkładki komorowe) |
| 2026 (maj–sierp.) | Wymiana węzła Kraszewskiego/Dąbrowskiego — eliminacja zwrotnic krzyżakowych z pkt 17 |
